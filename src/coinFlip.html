How to Play: Dice Roll

  1. Player can flip the coin as many times as they'd like, either once at a time, or in sets of multiple flips
  2. After each flip of the coin, the table will display which side the coin landed on
  3. When done flipping the coin, the player must guess if the coin is weighted or not based on the table results
  4. After the player makes their guess, if the coin is actually not weighted, then the weight will be displayed
  5. If the coin is actually weighted however, the player will be shown a prompt asking them to guess the weight, with a drop down contain all possible options, and a submit button to submit the answer
  6. To start a new game, start flipping the coin again
