Setting a Distribution

There are 3 options for choosing a distribution.

  1. Manually distribute chips

    2. This simply means adding and removing chips from cards using the +/- buttons.

    ADDING: You can use the blue + button to add a chip to a card.

    REMOVING: You can use the red - to remove a chip from a card.

  2. Randomly distributing chips
  3. Loading a saved distribution

Simulation Options

Number of Games - Choose the number of games that are run during one simulation. Stats will be tracked and recorded over the specified number of games.

Simulation Statistics

There are 2 areas to see statistics from recent simulations.

  1. Recent Games Table
  2. Recent Games Bar Graph
